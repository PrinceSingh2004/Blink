# Blink - Live Streaming & Social Interaction Platform

**Blink** is a full-stack web application built with **HTML, CSS, JavaScript, Node.js, and MySQL**. It allows users to go live, watch live streams, interact via live chat, and explore other users’ content in real-time.  

This project is designed for social interaction and real-time engagement.

---

## Features

- ✅ User authentication (Sign up / Login)  
- ✅ User profile management  
- ✅ Live streaming functionality  
- ✅ Live chat & commenting during streams  
- ✅ Browse other users’ live sessions  
- ✅ Interactive and responsive UI  
- ✅ MySQL database to manage users, streams, and messages  

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Real-time Communication:** Socket.io (for live chat & streaming notifications)  
- **Deployment (Recommended):** Render / Railway for backend, Vercel for frontend
